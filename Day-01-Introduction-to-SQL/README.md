# Day 01 - Introduction to SQL

## What is SQL?

SQL stands for Structured Query Language.

It is a standard language used to communicate with databases.

Using SQL we can:

- Store data
- Retrieve data
- Update data
- Delete data
- Manage databases

---

## What is a Database?

A database is an organized collection of data.

Examples:

### Student Database

| Student ID | Name | Branch |
|------------|------|---------|
| 101 | Chakri | AIML |
| 102 | Ram | CSE |
| 103 | Ravi | ECE |

The above table is stored inside a database.

## Types of SQL Commands

### DDL (Data Definition Language)

Used to create and modify database structures.

Commands:

- CREATE
- ALTER
- DROP
- TRUNCATE

---

### DML (Data Manipulation Language)

Used to work with data.

Commands:

- INSERT
- UPDATE
- DELETE

---

### DQL (Data Query Language)

Used to retrieve data.

Command:

- SELECT

---

### DCL (Data Control Language)

Used for permissions.

Commands:

- GRANT
- REVOKE

---

### TCL (Transaction Control Language)

Used for transactions.

Commands:

- COMMIT
- ROLLBACK
- SAVEPOINT

---

## SQL Workflow

User → SQL Query → Database → Result

Example:

SELECT * FROM students;

The database processes the query and returns all student records.

## Key Terms

### Database

Collection of related data.

### Table

Data stored in rows and columns.

### Row

Single record in a table.

### Column

Single field in a table.
